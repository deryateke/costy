# 🛍️ COSTY - Modern Shopping Cart Experience

COSTY is a responsive e-commerce web application built with **HTML**, **SCSS**, and **Vanilla JavaScript**. The project provides a smooth and user-friendly shopping experience where users can browse products, add items to their cart, manage quantities, and view the total amount dynamically.

Designed with simplicity and functionality in mind, COSTY demonstrates how modern front-end technologies can be used to create an interactive shopping platform without relying on external frameworks.

## ✨ Features

* Browse products fetched from a local JSON data source
* Add products to the shopping cart
* Increase or decrease product quantities
* Remove items from the cart with confirmation
* Dynamic cart counter with real-time updates
* Automatic total price calculation
* Delivery fee handling based on order amount
* Persistent cart data using Local Storage
* Responsive design for desktop and mobile devices
* Mobile navigation menu
* Custom "Cart Empty" page
* Modular SCSS architecture for maintainable styling

## 🛠️ Technologies Used

* HTML5
* SCSS (Sass)
* Vanilla JavaScript (ES6 Modules)
* Local Storage API
* Fetch API
* CSS Grid & Flexbox

## 📂 Project Structure

The application follows a modular architecture:

* **api.js** → Fetches product data
* **cart.js** → Handles cart operations
* **helpers.js** → Utility functions and calculations
* **ui.js** → Renders products and cart elements
* **SCSS modules** → Organized styles for header, products, cart, footer, and responsive layouts

## 🚀 Functionality

Users can:

* Explore available products
* Add items to the cart instantly
* Update quantities directly inside the cart
* Remove products whenever needed
* View the total amount dynamically
* Continue shopping with cart data preserved across page refreshes

## 🎨 Responsive Design

COSTY is fully responsive and optimized for different screen sizes. The layout adapts seamlessly to provide a consistent experience across desktop, tablet, and mobile devices.
